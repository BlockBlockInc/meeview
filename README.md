# Meeview

# Setup
1. Install dependencies: 
```
npm install 
```

2. Start application: 
```
npm start 
```

# Dependencies 
We rely on the following libraries to enable VRM usage:

1. https://github.com/pixiv/three-vrm
2. https://github.com/pmndrs/react-three-fiber
3. https://github.com/pmndrs/drei
